# BackgroundMute

Mute background applications while keeping foreground audio active.

## Install

```
winget install BackgroundMute
```

Or download from [releases](https://github.com/YourUsername/BackgroundMute/releases).

## Usage

Run the application. It automatically mutes audio from background processes while preserving sound from the active window.

## Requirements

Windows 10 or later with .NET Runtime 10.0.
